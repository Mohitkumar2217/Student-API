# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Mohitkumar2217/Golang.git
```

Go to the project directory

```bash
cd Golang
```

---

## Install Dependencies

```bash
go mod tidy
```

---

## Run Project

```bash
go run cmd/Students-api/main.go -config config/local.yaml
```

(depending on the project)

# 📷 Sample API

### Get Students

```http
GET /students
```

### Get Student

```http
GET /api/students/{id}
```

### Create Student

```http
POST /api/students
```

### Get all Student

```http
GET /api/students/{id}
```

### Delete Student

```http
DELETE /api/students/{id}
```
